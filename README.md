## How to Play

**Objective**: Try to get the highest score you can by avoiding obstacles!

**Controls**: 
* Press `SPACE` to make the bird jump
* Click the gear icon ‚öôÔ∏è to pause and access settings

## Features

* **Health System** - 3 lives (hearts) before game over
* **Scoring System** - Tracks your score as you pass obstacles
* **Options Menu** - Toggle music and sound effects
* **Pause Functionality** - Settings button to pause mid-game
* **Endless Gameplay** - Procedurally spawning obstacles
* **Smooth Physics** - Gravity and jump mechanics
* **Game Over Screen** - Shows final score with replay option
* **Responsive Controls** - Spacebar to flap

## Development Notes

* This is a learning project built to practice Unity game development
* Focused on core mechanics: physics, collision detection, and UI systems
* WebGL build optimized for browser performance

## Installation

To play locally:
1. Clone this repository
2. Open the WebGL build folder
3. Run with a local server (WebGL builds require a server to run properly)

Or simply play online at: [flappy-teal.vercel.app](https://flappy-teal.vercel.app)

## Known Issues

* Game requires a desktop browser with a keyboard
* Audio may have a delay on first play due to browser autoplay policies
